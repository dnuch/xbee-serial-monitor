# SPEARS/PHALANX Network UI
This application interfaces with nodes running on xbee mesh network
#### PACKAGE
- Angular 7
- serialport 7.1.4
- electron 4.0.5
- bootstrap 4
- node.js
- xbee-api
- angular5csv

#### PREREQUISITES
At root directory:

`
$ npm i
`
> if installing on windows; open administrative powershell and run: 
`
$ npm install --global windows-build-tools
`

`
$ npm run rebuild
`
### DEVELOPMENT
`
$ npm run electron
`
